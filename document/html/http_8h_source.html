40 // Find query parameter value of the given name.
41 //
42 // On failure, returns NULL.
43 // On success, returns an alias pointer to the value following name in the url,
44 // if `len_p` is not NULL, then stores the length of the value in `*len_p`.
45 //
46 // Example
47 // -------
48 // 1. http_get_param("/?key=val", "key", &len) will return a pointer to "val"
49 // and store 3 in len.
50 // 2. http_get_param("/?key&foo", "key", &len) will return a pointer to "&foo"
51 // and store 0 in len.
52 // 3. http_get_param("/?foo=bar", "key", &len) will return NULL and will not
53 // touch len.
