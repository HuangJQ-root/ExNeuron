Detailed Description

数据值联合体，用于存储不同类型的数据。

此联合体提供了多种数据类型的存储选项，包括基本数据类型（如整型、浮点型）、 字符串、字节数组、指针以及JSON对象等，适用于需要处理各种不同数据类型的情况。

Note
  • 各种类型数据共享访问空间：在modbus采集的使用中，采集的数据以字节形式按 顺序存在bytes中。如果标签类型是u16，则dvalue.value.u16 和 dvalue. value.bytes.bytes 共享同一块内存空间，所以可以直接通过 dvalue.value .u16 来访问之前复制的 2 字节数据
