Detailed Description

用于存储标签值及其元数据信息的结构体。

此结构体包含了与特定标签相关的所有必要信息，包括标签名称、标签值、 元数据数组和数据标签的详细信息。它在数据响应场景中使用，以完整地 描述一个数据点的所有相关信息。

Note

在数据响应场景中完整地描述一个数据点的所有相关信息。它是一个综合 性的数据结构，用于将标签名称、标签值、元数据以及数据标签的详细信 息整合在一起，方便在不同模块之间传递和处理与特定标签相关的完整数据。
