Detailed Description

缓存值结构体，用于存储标签值的相关信息。

此结构体包含了缓存中的数据值、获取或更新该值的时间戳以及与该值相关的元数据数组。 它被广泛应用于处理不同类型的数值及其相关的附加信息（如精度、单位等）。

Note

设计目的是在缓存中高效地存储和管理数据，提供数据的基本信息（如值、时间戳和元数据）， 以便快速访问和更新缓存中的数据。
