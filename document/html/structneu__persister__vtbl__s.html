Detailed Description

虚函数表，定义了一组函数指针，这些指针指向不同的函数， 这些函数实现了持久化对象的各种操作。通过使用虚函数表， 可以在运行时根据实际的持久化对象类型调用相应的函数， 从而实现多态性(不同的持久化对象如sqlite)。
