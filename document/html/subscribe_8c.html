处理订阅请求，将应用程序订阅到指定驱动程序的组。

该函数用于在订阅管理器中处理应用程序对指定驱动程序组的订阅请求。它会检查是否已经存在相同的订阅， 如果不存在则创建新的订阅条目并添加到订阅管理器中。

Parameters
mgr指向 neu_subscribe_mgr_t 结构体的指针，表示订阅管理器，包含订阅信息的哈希表等。
driver指向字符串的指针，表示要订阅的驱动程序名称。
app指向字符串的指针，表示发起订阅的应用程序名称。
group指向字符串的指针，表示要订阅的组名称。
params指向字符串的指针，表示订阅的参数，可为 NULL。
static_tags指向字符串的指针，表示静态标签，可为 NULL。
addrstruct sockaddr_un 类型的结构体，表示应用程序的地址。
Returns
返回订阅操作的结果状态码：
  • 如果内存分配失败（如 strdup 或 calloc 失败），返回 NEU_ERR_EINTERNAL。
  • 如果应用程序已经订阅了该组，返回 NEU_ERR_GROUP_ALREADY_SUBSCRIBED。
  • 如果订阅操作成功，返回 NEU_ERR_SUCCESS。
