neu_mqtt_client_publish_cb_t  cb

Publish a qos message with len bytes payload on topic.

This function initiates sending a PUBLISH packet, and if successful returns zero and the callback cb will be called once on completion of message delivery, otherwise returns a nonzero value. Callback invocation will be of the form cb(errcode, qos, topic, payload, len, data), where errcode is zero if delivery was successful and nonzero otherwise, and the other arguments are exactly the same what you pass into this function. You may set cb to NULL if you do not care about the result.
