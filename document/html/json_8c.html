解析JSON对象中的单个元素值到目标结构体

Parameters
rootJSON根对象（必须为JSON Object或Array类型）
ele目标元素描述结构体，定义要提取的字段名、类型和存储位置
Returns
int 解析结果：
  • 0: 成功（或可选字段不存在）
  • -1: 必填字段缺失或类型不匹配
Note
  • 支持自动类型推断（当ele->t为NEU_JSON_UNDEFINE时）
  • 处理字符串时会进行深拷贝（需调用方释放内存）
  • 数组类型会自动分配内存（需调用方释放）

解析JSON数据到预定义的结构体元素数组

Parameters
json输入的JSON数据（必须是已解析的json_t对象）
size目标元素数组的长度
ele目标元素数组指针
Returns
int 解析结果：
  • 0: 所有元素解析成功
