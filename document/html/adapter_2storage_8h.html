◆ adapter_load_group_and_tag()

int adapter_load_group_and_tag ( neu_adapter_driver_t *  driver)

从持久化存储中加载适配器驱动的组和标签信息。

该函数用于从持久化存储中读取与适配器驱动相关的组和标签信息，并将其添加到适配器驱动中。 它首先加载组信息，然后针对每个组加载相应的标签信息，并将这些信息添加到适配器驱动的内部结构中。

Parameters
driver指向 neu_adapter_driver_t 类型的指针，代表要加载组和标签信息的适配器驱动实例。
Returns
函数的返回值为整数类型。
  • 如果在加载组或标签信息的过程中出现错误，将返回相应的错误代码（非零值）， 并记录一条警告日志，提示加载失败的具体信息。
  • 如果所有组和标签信息都成功加载，将返回 0。

char **  setting

从持久化存储中加载适配器的设置信息。

该函数通过调用 neu_persister_load_node_setting 函数，从持久化存储中 读取指定适配器的设置信息。如果加载成功，设置信息将存储在由 setting 指向 的字符串指针所指向的内存中。

Parameters
node指向表示适配器名称的字符串指针，用于指定要加载设置的适配器。
setting指向字符指针的指针，用于存储加载的适配器设置信息。
Returns
函数的返回值表示加载操作的结果。
  • 若返回 0，表示设置信息加载成功。
  • 若返回 -1，表示设置信息加载失败，此时会记录一条警告日志，提示加载指定适配器设置失败。
