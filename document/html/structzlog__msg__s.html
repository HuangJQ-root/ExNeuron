Detailed Description

日志消息结构体定义。

该结构体用于存储和传递日志消息的相关信息，包括：

  • buf: 指向包含日志消息数据的缓冲区。
  • len: 缓冲区中的数据长度。
  • path: 日志路径或标识符，用于确定日志的优先级。
