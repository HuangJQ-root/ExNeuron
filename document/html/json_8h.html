◆ neu_json_attribute

enum neu_json_attribute
Enumerator
NEU_JSON_ATTRIBUTE_REQUIRED 

表示该JSON元素是必需的。

如果设置了NEU_JSON_ATTRIBUTE_REQUIRED，则在解析JSON时， 必须找到对应的键，否则视为错误。

NEU_JSON_ATTRIBUTE_OPTIONAL 

表示该JSON元素是可选的。

如果设置了NEU_JSON_ATTRIBUTE_OPTIONAL，则在解析JSON时， 该键可以不存在，不会导致错误。

解析JSON数据到预定义的结构体元素数组

Parameters
json输入的JSON数据（必须是已解析的json_t对象）
size目标元素数组的长度
ele目标元素数组指针
Returns
int 解析结果：
  • 0: 所有元素解析成功
  • -1: 输入参数无效或解析过程中出错
Note
  • 要求调用方提前解析好json_t对象（如通过json_loads）
  • 按顺序解析ele数组中的每个元素，遇到第一个错误立即返回
  • 需要调用方负责最终释放动态分配的内存（如字符串、数组）
