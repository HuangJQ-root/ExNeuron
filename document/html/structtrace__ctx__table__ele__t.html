Detailed Description

定义用于存储追踪上下文表元素的结构体。

trace_ctx_table_ele_t 结构体用于在哈希表中存储每个 请求上下文（key）及其对应的追踪上下文（ctx）。 hh 是 uthash 库使用的特殊字段，用于管理哈希表中的元素。
