Detailed Description

结构体用于封装IO事件的相关参数。

该结构体包含触发IO事件的文件描述符、用户自定义数据以及事件触发时要调用的回调函数。 需要 fd 成员来标识触发IO事件的文件描述符

Member Data Documentation

◆ cb

neu_event_io_callback neu_event_io_param::cb

当IO事件触发时调用的回调函数。

这是一个函数指针，指向一个回调函数。当IO事件发生时，这个回调函数会被调用。 回调函数的签名应该与neu_event_io_callback类型相匹配。

触发IO事件的文件描述符。

这是一个整型变量，代表在UNIX和类UNIX系统中用于标识打开文件 或套接字等非文件资源的文件描述符。
